# 📊 Pulse-PM — Agile Project Management & Analytics Dashboard

Pulse-PM is a high-fidelity, interactive Project Management Dashboard designed to streamline Agile and Scrum workflows. It bridges the gap between structured performance metrics and elegant user experience design, providing project managers and business analysts with crystal-clear project lifecycle tracking.

---

## 🚀 Key Features

* **Interactive Kanban Board**: Seamless drag-and-drop task lifecycle mapping (To Do ➡️ In Progress ➡️ Done) with asynchronous visual column counters.
* **Real-Time Activity Stream**: Localized state tracking logs every transaction, workspace creation, and lane transition with instantaneous timestamping.
* **Contextual Node Management**: Project cards feature an embedded 3-dots action dropdown with strict propagation isolation for on-the-fly modifications.
* **Advanced Reports & Analytics**: Dynamic rendering of completion velocity rates through custom CSS-driven bar charts and variable conic-gradient donut graphs.
* **Premium Lavender Aesthetic**: Carefully mapped UI theme using flat, calming lavender layouts designed to minimize user screen-fatigue during sprints.

---

## 🛠️ Tech Stack Used

* **Frontend Architecture**: Clean Semantic HTML5, Modular CSS Grid/Flexbox Layouts
* **Logic & Interactions**: Native Vanilla JavaScript (ES6+ Web APIs & Event Handlers)
* **Authentication Engine**: Firebase Auth Integration
* **Data Persistence Framework**: Structured LocalStorage JSON Repositories

---

## 📂 Project Structure

* `index.html` — User authentication portal (Login / Signup)
* `dashboard.html` — Main tracking dashboard and analytical views
* `dashboard-style.css` — Core typography layout and lavender theme design
* `dashboard-script.js` — Kanban board engine and data visualization controller
* `style.css` & `script.js` — Core helper utilities
